# DevIAR Web

DevIAR Web is a personal service landing page focused on web development, AI automation, and high-performance digital solutions.
It is designed as a lead-generation website that presents services, background, selected projects, and direct contact paths.

## Overview

- Personal brand landing page with a conversion-focused structure.
- Presents web development and AI automation services.
- Includes hero messaging, personal story, projects, and contact section.
- Built as a static site optimized for GitHub Pages deployment.
- Includes SEO, Open Graph, and structured data metadata.

## Core Highlights

- Strong hero section with service positioning.
- Mobile navigation and responsive layout.
- Read-more sections for storytelling without overwhelming the page.
- Technical service framing around performance, SEO, automation, and UX.
- Contact flow connected to FormSubmit.
- Security-oriented metadata including Content Security Policy.

## Local Setup

This is a static project, so you can open it directly or serve it locally.

### Option 1

Open `index.html` in your browser.

### Option 2

Serve it with a local static server:

```bash
npx serve .
```
